# Guide: WebP and Core Web Vitals

Images are usually the heaviest thing on a page, so switching to WebP is one of the fastest ways to improve Largest Contentful Paint. Convert your hero and product images at a quality around 75–85 for photos — you'll often cut a third off the file size with no visible difference.
