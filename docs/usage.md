# Usage & API

It is a standard custom element, so it works with no wrapper in plain HTML, React, Vue, Svelte and Astro.

## Plain HTML

```html
<script src="webp-converter.js"></script>
<webp-converter></webp-converter>
```

## React

```jsx
import "@sgbp/webp-converter";
export default function Page() { return <webp-converter />; }
```

## Vue

```vue
<script setup>
import "@sgbp/webp-converter";
</script>

<template>
  <webp-converter />
</template>
```
