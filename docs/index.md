# Image to WebP Converter

Convert PNG, JPG or GIF to WebP right in your browser — nothing is uploaded. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="webp-converter.js"></script>
<webp-converter></webp-converter>
```

## What it does

WebP is a modern image format that compresses 25–35% smaller than JPEG or PNG at the same quality, which makes pages load faster. This converter does it client-side with the Canvas API — your image never leaves your device — and shows the size saving with a quality slider.

## When you'd use it

Shrinking hero and product images before you upload them, or batch-modernising an old PNG/JPEG library.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
